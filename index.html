Git & GitHub

Version control basics — track, save, and share your code.

What's the difference?

🛠 Git

Runs on your computer. Tracks every change you make to your files.

☁️ GitHub

A website to store and share your Git projects in the cloud.

How it works

1

Edit files

Make changes in your project folder as usual.

2

git add

Choose which changes you want to save.

3

git commit

Take a snapshot of your project with a message.

4

git push

Upload your snapshot to GitHub.

Key commands

git initStart a new repo
git clone <url>Download from GitHub
git statusSee what changed
git add .Stage all changes
git commit -m ""Save a snapshot
git pushUpload to GitHub
git pullGet latest changes
git logView history
